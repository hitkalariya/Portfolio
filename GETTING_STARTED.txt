GETTING STARTED - Hit Kalariya Portfolio

Follow these 6 commands to get the project running locally:

1. git clone https://github.com/hitkalariya/portfolio.git
2. cd portfolio
3. cp .env.example .env.local
4. pnpm install
5. pnpm db:push && pnpm db:seed
6. pnpm dev

The application will be available at http://localhost:3000

IMPORTANT: After step 3, edit .env.local with your actual API keys:
- Set up Cloudinary account for media storage
- Get Gemini API key from Google AI Studio
- Configure Resend for email functionality
- Add GitHub personal access token
